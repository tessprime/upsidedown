```
    std::string x = "When it feels like all the world is wearing a frown,";
    std::string y = "Put a smile on and spread it around!";
    std::string z = "With your smile turn the world upside down!";

    std::string x = "'uʍoɹɟ ɐ ƃuᴉɹɐǝʍ sᴉ plɹoʍ ǝɥʇ llɐ ǝʞᴉl slǝǝɟ ʇᴉ uǝɥM";
    std::string ʎ = "¡punoɹɐ ʇᴉ pɐǝɹds puɐ uo ǝlᴉɯs ɐ ʇnԀ";
    std::string z = "¡uʍop ǝpᴉsdn plɹoʍ ǝɥʇ uɹnʇ ǝlᴉɯs ɹnoʎ ɥʇᴉM";
```

A ridiculous c++ source to source transformer that makes user defined symbols and stirng literals... upsidedown.

# Building
```
mkdir build
cd build
cmake ..
make
```

## Usage
```
./build/upsidedown laurie.cpp
```
