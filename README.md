# Upsidedown

When it feels like all the world is wearing a frown,
Put a smile on and spread it around!
With your smile turn the world upside down!

[![Black Gryph0ns Upside down](https://img.youtube.com/vi/sJ0yPYUnLlE/0.jpg)](https://www.youtube.com/watch?v=sJ0yPYUnLlE)

# Building
```
mkdir build
cd build
cmake ..
make
```

## Usage
```
./build/upsidedown laurie.cpp
```
